# Roadmap: TIEV Global Website

## Overview

Transform the existing SoapBox marketing demo into a premium website for TIEV missionary organization. The work progresses from branding updates through content pages to final polish, leveraging the existing design system throughout.

## Domain Expertise

None

## Phases

- [ ] **Phase 1: Branding & Navigation** - Replace SoapBox branding, update Navbar/Footer for TIEV
- [ ] **Phase 2: Home Page** - Hero, mission statement, impact overview
- [ ] **Phase 3: About Page** - Organization history, values, team
- [ ] **Phase 4: Project Pages** - Iraq, Pakistan, Philippines, Japan country pages
- [ ] **Phase 5: Donate & Contact** - Donation page with CTAs, contact form
- [ ] **Phase 6: Polish & Launch** - Final cleanup, SEO, deploy

## Phase Details

### Phase 1: Branding & Navigation
**Goal**: Replace all SoapBox branding with TIEV identity and update navigation structure
**Depends on**: Nothing (first phase)
**Research**: Unlikely (internal patterns, existing components)
**Plans**: 2 plans

- [ ] 01-01: Update Navbar with TIEV logo, navigation links, and mobile menu
- [ ] 01-02: Update Footer with TIEV branding, links, and social media
